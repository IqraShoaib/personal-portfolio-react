# Personal Portfolio Repository

A full-stack portfolio repository containing multiple projects, including frontend, backend, and a React-based portfolio application. This repository demonstrates modern web development practices, responsive design, reusable components, API integration, and performance optimization.

---

# Repository Structure

```text
.
├── frontend/
│   ├── Source code for the frontend application
│   └── Assets and components
│
├── backend/
│   ├── Server-side application
│   ├── APIs
│   └── Data handling
│
└── portfolio-react/
    ├── React portfolio application
    ├── Components
    ├── Assets
    ├── Public
    └── Vite configuration
```

# Installation

Clone the repository

```bash
git clone https://github.com/IqraShoaib/personal-portfolio-react.git
```

Move into the project

```bash
cd personal-portfolio-react
```

Install dependencies for the React application

```bash
cd portfolio-react
npm install
```

Run the development server

```bash
npm run dev
```
